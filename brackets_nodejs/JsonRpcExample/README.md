여덟 번째 노드 프로젝트인 JsonRpcExample 에서는  
  
1. JSON-RPC를 웹 서버에 적용하기.  
2. 계산기 모듈을 추가하여 실행하기.  
3. 데이터베이스에서 자용자 리스트 조회하기.  
4. 데이터 부분을 암호화하기.  
  
을 공부 했습니다.
